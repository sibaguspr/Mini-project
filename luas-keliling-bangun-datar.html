Luas segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Dimana :

L = Luas

a = Alas

t = Tingi

Keliling segitiga

Rumus Keliling Segitiga yaitu :

K = s1 + s2 + s3

Dimana :

K = Keliling

s1 = sisi A

s2 = sisi B

s3 = sisi C

Hitung Keliling Segitiga

Luas Jajargenjang

a = Alas

t = Tinggi

Keliling Jajargenjang
